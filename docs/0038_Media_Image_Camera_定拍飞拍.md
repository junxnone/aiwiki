# Brief

模式 | Description | 优缺点
-- | -- | --
**定拍** | 轴运动到拍照位停下来相机拍照处理，然后把结果反馈给执行机构 | 有延时, 某些环境对延时要求高，不适合该模式<br>精度高
**飞拍** | 轴运动到拍照位时运动不停完成拍照处理以及结果反馈 | 不需要停滞<br>精度比 定拍稍低
