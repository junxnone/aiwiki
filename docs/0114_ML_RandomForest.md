# Random Forest 随机森林

## Reference
- [随机森林 – Random forest](https://easyai.tech/ai-definition/random-forest/)

## Brief
- Random Forest - 随机森林
- 由很多决策树构成，不同决策树之间没有关联
  - 属于 集成学习 中的 Bagging
- **优点**
  - 不容易过拟合
  - 训练速度比较快, 可并行
- **缺点**
  - 对含噪音数据会过拟合


![image](https://user-images.githubusercontent.com/2216970/112955448-03bdc600-9172-11eb-9794-463f0768b79e.png)

## UseCase

- 对离散值的**分类**
- 对连续值的**回归**
- 无监督学习**聚类**
- **异常**点检测
