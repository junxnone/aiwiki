## Brief

Name | Description
-- | --
法曲率(Normal curvature) | 曲面在一点沿不同方向的曲率
主曲率 (Principal Curvatures) | 法曲率中的最大值 $k_1$ 和最小值 $k_2$
平均曲率(Mean Curvature) | 曲率在空间中的弯曲程度 $H=\frac{k_1+k_2}{2}$
高斯曲率(Gaussian Curvature) | 内在弯曲程度 $K = k_1\times k_2$
