# AIGC

- AIGC: `AI Generated Content` 利用 AI 生成内容 - 绘画/写作/...
  - PGC: `Professional Generated Content` 专业平台或组织生成的内容， 电视/电影/海报/网剧/...
  - UGC: `User Generated Content` 用户个体生成的内容，个人vlog/blog/...


```mermaid

graph TD
    T(Text) --> A(Audio)
    T --> I(Image)
    I --> I
    T --> D(3D)
    T --> V(Video)
    I --> T
    T --> C(Code)
    T --> T
    T --> S(Science)
```
