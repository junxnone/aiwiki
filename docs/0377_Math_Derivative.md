# Derivative 导数

- 函数在某一点的导数是指这个函数在这一点附近的变化率
- $\displaystyle \lim_{ x \to a}\frac{f(x) - f(a)}{x-a}$
- $\displaystyle \lim_{ \Delta x \to 0}\frac{f(x_{0} + \Delta x) - f(x_{0})}{\Delta x}$
