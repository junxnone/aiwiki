# Derivative 导数

- 函数在某一点的导数是指这个函数在这一点附近的变化率
- $\displaystyle \lim_{ x \to a}\frac{f(x) - f(a)}{x-a}$
- $\displaystyle \lim_{ \Delta x \to 0}\frac{f(x_{0} + \Delta x) - f(x_{0})}{\Delta x}$
- $\frac{df}{dx} |_{x=x_{0}}$
- ${f}\'(x_{0})$ 

## 一元函数 - 导数

- f(x)  在 x 轴正方向的变化率

## 多元函数 - 偏导数

- 函数 $f(x_{1},x_{2},x_{3}, ...)$ 在某一坐标轴 $(x_{1},x_{2},x_{3}, ...)$ 上的变化率


## 方向导数

- 某一点在某个方向上的导数值
