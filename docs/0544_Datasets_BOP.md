# BOP
- BOP - Benchmark for 6D Object Pose Estimation - 6D 物体位姿估计数据集
- 用于 6D 姿态 / 位姿检测、机器人抓取、工业视觉
- 一般提供如下数据: 
  - RGB-D图像
  - 物体3D模型
  - 6D真实位姿(GroundTruth)
  - 掩膜/边框标注



## 1. lmo.tar.gz → LM-O (Linemod-Occluded，遮挡版Linemod)
- **物体**：8个日常小件（杯子、罐头、订书机等），源自经典Linemod
- **特点：重度遮挡场景**，画面里物体大面积被其他杂物遮挡，是最经典入门遮挡测试集；
- **用途**：通用物体遮挡场景6D位姿基准，论文必测标配。

## 2. ycbv.tar.gz → YCB-Video(YCB-V)
- **物体**：21种家用日用品（塑料玩具、工具、食物容器），YCB工业标准物件；
- **特点：动态视频序列采集、杂乱桌面、光照多变、多物体堆叠，附带海量合成训练数据**，机器人抓取领域最主流数据集（PoseCNN提出时使用）；
- **难度：中等偏难，真实居家杂乱环境，工业落地首选评测集**。

## 3. tless.tar.gz → T-LESS（无纹理工业零件数据集）
- **物体**：30个**无纹理、类对称、外形高度相似的工业五金零件**（螺丝座、金属卡扣），几乎无表面纹理；
- **核心难点：无纹理+几何对称极易造成角度跳变（就是你遇到的±180°旋转误差来源）**，工业质检最难数据集之一；
- **场景：多零件密集堆叠，大量自遮挡**。

## 4. itodd.tar.gz → ITODD(MVTec工业零件数据集)
- **物体**：28个高精度金属工业零部件（MVTec公司出品，精密机械配件）；
- **特点：金属强反光、镜面高光、物体几何对称、工厂环境复杂光照**，面向自动化产线质检；
- **规模：实景图片偏少，侧重严苛工业反光场景测试**。

## 5. icbin.tar.gz → IC-BIN（工业料盒分拣数据集）
- **物体**：注塑塑料料盒、工业收纳盒，外形规整、部分轴对称；
- **场景：料箱杂乱堆放、箱体互相嵌套遮挡，模拟自动化料仓分拣场景**；
- **难点：纯色塑料+对称结构，FP16量化后旋转出错高发。**

## 6. hb.tar.gz → HB(Homebrewed Dataset，HB自制家居杂物集)
- **物体**：日常家用杂乱小件（瓶子、厨具、小五金）；
- **特点：近距离拍摄、物体密集重叠、复杂室内自然光，偏生活化杂乱场景**，侧重非规整日用品位姿。

## 7. tudl.tar.gz → TUD-L（TUD工业灯具零件数据集）
- **物体**：照明灯具塑料/金属配件，大量圆柱、轴对称结构零件；
- **场景：工厂工作台散乱摆放，反光+对称双难点，工业自动化装配专用评测集**。

# 补充（贴合你FP16推理异常）
> **TLESS/ITODD/TUDL/ICBIN是对称/无纹理物体**，FP16半精度量化很容易出现180°旋转偏差，FP32全精度数值冗余多不容易翻转，这就是你测试出现角度差180°的行业共性问题。
