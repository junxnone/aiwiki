## Brief
- Dynamic Programming - 动态规划
- 记住已经解决过的子问题的解
- **备忘录法**: 自顶向下递归
- **动态规划**: 自底向上
  - 重叠子问题
  - 最优子结构
- 线性模型
- 区间模型
- 背包模型
- 树形模型
