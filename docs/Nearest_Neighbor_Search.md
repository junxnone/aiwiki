# Brief
- NNS - `Nearest Neighbor Search`
- **问题** : `M` 空间内的点集 `S` 和点 `q`，查找 `S` 中离 `q` 最近的点
  - 1973 **post-office problem** `分配一个地址到最近的邮局`
- 精确搜索
  - 线性搜索 - 遍历数据集
  - 空间分割 - [KDTree](https://github.com/junxnone/tech-io/issues/1036)
- 近似搜索

## 延伸问题
- KNN - 预测数据分类
- [ANN](/Approximate_Nearest_Neighbor) - 近似最近邻
