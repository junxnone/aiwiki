## Brief

  - FLANN - Fast Library for Approximate Nearest Neighbors
  - 用于在高维空间中搜索最近邻
      - **最近邻搜索问题**: `在度量空间X给定一组点P=p1,p2,…,pn, 查询点q属于X, 在P中找到距离q最近的点的问题`
  - Support Languages - C++/C/MATLAB/Python/Ruby
  - OpenMP support
