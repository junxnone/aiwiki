## 激光测距

  - ToF
      - 脉冲测量
      - 相位测量
  - 空间几何
      - 三角测量
      - 干涉测量

| 方法   | 适合距离     | 精度 | 应用场景            |
| ---- | -------- | -- | --------------- |
| 脉冲测量 | 长距离      | m  | 遥测/地质勘探/飞行器/... |
| 回波测量 |          |    |                 |
| 相位测量 |          | mm |                 |
| 三角测量 | \<2000mm | um |                 |

### 三角测量

  - 随着被测物体距离的改变，光线聚焦到感光元件的位置不同

| 原理 | **D = c x △t/2** |
| -- | ---------------- |
| D  | 被测距离             |
| c  | 光速               |
| △t | 测试时间             |

![cj](media/7df630ffc3037419b878bc025c0d01d2d5b1b847.gif)

### 相位测量

| 原理 | D = c x △t = c x △φ/ 4πf = λ/2 x △φ/2π |
| -- | -------------------------------------- |
| D  | 被测距离                                   |
| c  | 光速                                     |
| △t | 测试时间 `△t=△φ/2πf`                       |
| △φ | 相移                                     |
| f  | 调制频率                                   |
| λ  | 调制波长 `λ=c/f`                           |
