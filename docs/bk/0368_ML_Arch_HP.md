# Hyperparameter 超参数

  - batch size
  - LR
  - Optimizer parameters
  - ...

## Batch Size

  - 大多数超参数的最佳值对Batch Size敏感, 因此, 更改 Batch Size 通常需要重新开始调整过程。
  - 尽量最开始选择合适的 Batch Size, 否则后期调整，需要调整其他超参，费时费力。
