# 多模态大模型

  - LLMs + 多种数据类型 =\> LMMs
  - `有些模型，如 Midjourney、Stable Diffusion 和 Dall-E，虽然可以处理多种数据，但并没有集成语言模型。`
  - Task
      - 生成
          - 文生图
          - 文字生成(输入为文字/图像)
      - VLU 视觉语言理解
          - 图像分类
          - 基于文本的图像检索
  - 训练
      - CLIP
      - Flamingo
