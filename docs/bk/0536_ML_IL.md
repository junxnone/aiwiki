# 增量学习/持续学习

  - Incremental Learning - Lifelong Learning - Continuous Learning -
    Never Ending Learning
  - 新增数据时，只做关于新增数据的更新，增量学习可以不断从新样本中学习新的知识，同时保存大部分以前学习到的知识

## 方案

  - 记忆回放（Replay / Rehearsal）
  - 参数正则化（EWC, SI, MAS）
  - 动态架构 / 模块化（专家混合）
  - 知识蒸馏（Knowledge Distillation）
