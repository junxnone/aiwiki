# Uformer

  - Uformer 是中科大等提出的通用 U 型 Transformer 图像复原模型，把 CNN 的 U-Net 结构和
    Transformer 的全局建模能力融合，主打去噪、去雨、去模糊、去反光 / 炫光等低层视觉任务。
