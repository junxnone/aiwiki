## Brief

Name | Description
-- | --
法曲率(normal curvature) | 曲面在一点沿不同方向的曲率
主曲率 (principal curvature) | 法曲率中的最大值和最小值
平均曲率 | 曲率在空间中的弯曲程度
高斯曲率 | 内在弯曲程度
