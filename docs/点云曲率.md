## Brief

Name | Description
-- | --
法曲率(normal curvature) | 曲面在一点沿不同方向的曲率
主曲率 (principal curvature) | 法曲率中的最大值 $k_1$ 和最小值 $k_2$
平均曲率 | 曲率在空间中的弯曲程度 $H=\frac{k_1+k_2}{2}$
高斯曲率 | 内在弯曲程度 $K = k_1\times k_2$
